## 关于栅格系统
栅格系统是Bootstrap的核心，其中精妙的思想将网页划分为12列，再按屏幕大小，分为sm(超小屏幕768px, md(中等屏幕992px), xs(大屏1200px), lg(超大屏)。通过形如col-md-x及col-md-offset-x组合使用，让布局不再复杂。
